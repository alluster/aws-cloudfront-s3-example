## AWS CloudFront + S3 deployment example

This TypeScript boilerplate was deployed using the AWS S3 connected tp AWS CloudFront.

## Authors

-   [@alluster](https://www.github.com/alluster)

## Installation

You need:
Node.js

```bash
$ npm install
```

## Running locally

```bash
$ npm run start
```

To view UI of the server navigate to:

```bash
localhost:3000
```
